2. Programmation des boutons


Insérer entre les balises <head></head> l'instruction en javascript de la fonction afficher() . Cette fonction permet d'afficher dans l'élément input  matérialisé par un formulaire, une valeur envoyée par  l'élément interactif button. 

      <script> 
         //fonction qui affiche la valeur
         function afficher(val) 
         { 
             document.getElementById("output").value+=val; 
         } 
      </script>


Rajouter le gestionnaire l'évènement onclick   sur l'élément  button du chiffre 1. L'événement saisi bouton 1, déclenche la fonction afficher() 

<td><button onclick="afficher()">1</button></td> 
À toi de jouer 2.1 :

À l'exception du bouton c, rajouter aux autres boutons l'événement onclick, afin d'afficher leurs valeurs respectives.

Le rendu doit être le suivant : lorsque je clique sur un bouton, sa valeur apparaît sur le formulaire d'affichage. (Tester par vous-même)



Insérer entre les balises <head></head> l'instruction en javascript de la fonction effacer() : Cette fonction permet d'effacer le contenu du formulaire d'affichage. 

      <script> 
         //fonction qui efface l'écran 
         function effacer() 
         { 
             document.getElementById("output").value = "";
         } 
      </script>



À toi de jouer 2.2 :

Rajouter l'évènement onclick  sur l'élément  button c (clear).

<td><button onclick="effacer()">c</button></td> 

Le rendu doit être le suivant : lorsqu'on clique sur le  bouton c, efface le contenu afficher dans le formulaire s'efface. (Tester par vous-même)



Insérer entre les balises <head></head> l'instruction en javascript de la fonction calculer() : Cette fonction permet de calculer les opérations saisie le contenu du formulaire d'affichage.

      <script> 
         //La fonction eval() permet d'évaluer du code JavaScript représenté sous forme d'une chaîne de caractères.
         function calculer() 
         { 
             let a = document.getElementById("output").value;
             let b = eval(a); 
             document.getElementById("output").value = b; 
         } 
      </script>


À toi de jouer 2.3 :

Rajouter l'évènement onclick  sur l'élément  button c (clear).

<td><button onclick="effacer()">c</button></td> 

Le résultat attendu : Une calculatrice simple parfaitement opérationnelle. (Tester par vous-même)




3. Événement 'onload'

Dès le chargement de la page,  il est  possible d'afficher un message  dans un popup. 

Pour cela, il vous suffit de rajouter l’attribut onload dans la balise <body>. 

onload = intro()

À toi de jouer 3.1 :

Faire afficher, dès le chargement de votre page, un message (d'introduction par exemple). 

Le script de la fonction intro()
      <script> 
         //La fonction intro() affiche un message dans une boîte de dialogue
         function intro() 
         { 
             alert("Message d'introduction");
         } 
      </script>

Type de résultat attendu dès le chargement de la page.


4. Événement 'onfocus'


onfocus modifie la couleur d'arrière-plan d'un champ de saisie lorsqu'il obtient le focus.

Pour cela, il vous suffit de rajouter l’attribut onfocus dans la balise <input>. 

onfocus="couleur(this)"

À toi de jouer 4 .1 :

Modifier avec la fonction couleur(), en 'bleu', la couleur de l'arrière-plan du formulaire  d'affichage des résultats (ou de saisie des opérations).  

Le script de la fonction couleur() 
      <script> 
        //fonction qui modifie la couleur de fond du formulaire
         function couleur(x) {
            x.style.background = "couleur_désirée";
          } 
      </script>
